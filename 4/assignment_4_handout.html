inertia = []  # List to store inertia values for each k

Task 1: Answer the following questions¶

  1. Do clustering algorithms (like $k$-means) take into account the number of classes or the value of the labels in a dataset? No, that information is unknown to the algorithms. And as unsupervised learning techniques, they try to find any sort of structure in the dataset, which may not result in the classes we know.

  2. How might variations in handwriting, such as different writing styles, pen types, or stroke patterns, affect the clustering of digits? Can a single class (e.g., the digit '8') appear in more than one cluster? The variations within a class of digits can greatly affect the clustering if the dissimilarity between handwritings is greater than the dissimilarity between different digits of the same handwriting. If the algorithm finds a greater dissimilatity between different handwritings (rather than different digits), it can place the same digit in more than one cluster.

  3. Can one cluster contain data points from multiple classes (e.g., digits '5' and '8')? Definitely, similar digits (like 5 and 8) may end up the same clusters. Even with enough clusters to cover all classes, the dissimilarity between their features can be lower than the dissimilarity between different handwritings.

  4. Is the elbow method an entirely objective way to determine the optimal number of clusters $k$? No, the method is subjective.

def evaluate_pca_accuracy(X_train, Y_train, X_val, Y_val, n_components_list):
    """Evaluate accuracy of Logistic Regression with different numbers of PCA components."""
    train_accuracies = []
    val_accuracies = []

    for n_components in n_components_list:
        # Apply PCA
        pca = PCA(n_components=n_components)
        X_train_pca = pca.fit_transform(X_train)
        X_val_pca = pca.transform(X_val)

        
        # Train Logistic Regression model
        clf = LogisticRegression(random_state=42, max_iter=1000)
        clf.fit(X_train_pca, Y_train)

        train_pred = clf.predict(X_train_pca)
        val_pred = clf.predict(X_val_pca)

        train_accuracy = accuracy_score(Y_train, train_pred)
        val_accuracy = accuracy_score(Y_val, val_pred)

        train_accuracies.append(train_accuracy)
        val_accuracies.append(val_accuracy)

    return train_accuracies, val_accuracies
